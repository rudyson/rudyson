# Config for [Mozilla Firefox](https://www.mozilla.org/uk/firefox/new/)

1. Type `about:support` in search bar
2. Find profile folder (`about:profiles`). It should be `Default Profile:yes` Root Directory
3. Close all opened browser instances
4. Paste `user.js` content in the end of original file in your profile folder
5. Save file
6. Reboot browser

See also explanation how it works:
https://www.youtube.com/watch?v=EGluCuXPL48
